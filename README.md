# Hardware-Bestandsverwaltung

Mein erstes größeres C# Projekt für die Weiterbildung zum .NET Developer.

## Was macht das Programm?

Das ist eine Konsolenanwendung mit der man IT-Geräte verwalten kann. Man kann Laptops, PCs, Server und Tablets hinzufügen, bearbeiten und löschen. Die Daten werden in JSON-Dateien gespeichert.

## Warum habe ich das gemacht?

Das ist mein Abschlussprojekt für den C#-Entwickler Kurs. Ich wollte zeigen was ich gelernt habe:
- Objektorientierte Programmierung
- Mit Dateien arbeiten (JSON)
- Menüs in der Konsole
- Fehlerbehandlung

## Was kann das Programm?

- Neue Geräte anlegen (Laptop, PC, Server, Tablet)
- Geräte bearbeiten (alle Eigenschaften änderbar)
- Geräte löschen
- Alle Geräte anzeigen
- Status ändern (Aktiv, Inaktiv, Defekt, Bestand)
- Automatische Speicherung in JSON-Dateien

## Was habe ich benutzt?

- C# (.NET 9)
- Visual Studio 2026
- Newtonsoft.Json für JSON

## Wie startet man das Programm?

### Voraussetzungen
- .NET 9 SDK installiert haben
- Visual Studio oder VS Code
